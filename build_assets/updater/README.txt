IntenseRP Next Updater - Windows
================================

QUICK START
1. Run IntenseRP-Updater.exe
2. Follow the on-screen prompts to install or update IntenseRP Next

FEATURES
- Install IntenseRP Next to any directory
- Update existing installations while preserving save data
- Check latest release information
- Automatic backup and restore of user data during updates

SYSTEM REQUIREMENTS
- Windows 10 or Windows 11
- Internet connection for downloading updates
- A little bit of patience

USAGE
Simply double-click IntenseRP-Updater.exe to launch the updater.
The updater will guide you through the process with a colored CLI interface.
